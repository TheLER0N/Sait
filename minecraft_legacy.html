О проекте

Глобальный проект по ремейку Legacy-версии Minecraft. Воссоздание атмосферы и геймплея классической консольной версии с использованием современных технологий.

Реализована кастомная генерация миров, поддержка загрузки Bedrock-миров и P2P сервера для мини-игр. Написано с использованием современных графических API (Vulkan/bgfx).

Каждый блок, каждый биом — воссозданы с любовью к оригиналу, но с улучшенной производительностью и современным рендерингом.
